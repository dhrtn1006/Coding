사용 언어 : Javascript

풀이 내용 :

```
function solution(d, budget) {
    let answer = -1;
    
    d.sort((a, b) => a - b);
    
    while (budget >= 0) {
        budget -= d[0];
        d.shift();
        answer++;
    }
    
    return answer;
}
```
